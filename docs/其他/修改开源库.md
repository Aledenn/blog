###使用

#### 生成补丁

```shell
npx patch-package preact --exclude '^$' 
// 默认不会记录package.json，所以使用--exclude覆盖
```

#### 使用补丁

```shell
git apply --ignore-whitespace patches/preact+10.4.7.patch
```

#### 撤销所有修改

```shell
npx patch-package --reverse 
```



### 优势

1. 可以快速的解决一些开源库的bug
2. 可以减少复制粘贴的工作量
